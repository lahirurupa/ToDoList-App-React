# 📝 To-Do List App

A simple and user-friendly To-Do List app built using **React**. This app helps users manage their daily tasks efficiently with options to add, delete, and prioritize tasks.

## 🚀 Features

- ✅ **Add Tasks** – Users can add new tasks to the list.
- 🗑 **Delete Tasks** – Remove tasks that are no longer needed.
- 📌 **Prioritize Tasks** – Move important tasks to the top and less important tasks to bottom.
- 🎨 **Responsive UI** – Clean and modern design with dark theme.
- 💾 **Local Storage** – Tasks persist even after refreshing the page.
